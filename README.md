Personal Expense Tracker

This project is built with plain HTML, CSS, and JavaScript.

Files included:
index.html
styles.css
script.js
README.md

How to run:
Open index.html in any modern browser.

Features:
Add, edit, and delete expenses.
Validate title, amount, category, and date.
Filter by category and date range.
Sort by date, amount, or category in ascending or descending order.
Show the total for the currently displayed expenses.
Store data in the browser using localStorage.

No installation or build step is required.
